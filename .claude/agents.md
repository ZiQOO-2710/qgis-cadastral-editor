# 에이전트 역할 정의

## Context7 Agent

**역할**: 라이브러리 문서 검색 전문가

**활성화 조건**:
- "사용법", "API", "문서", "예제", "라이브러리" 키워드
- QGIS/PyQGIS 관련 질문
- Leaflet, Python 패키지 사용법

**주요 기능**:
- QGIS/PyQGIS API 문서 검색
- Leaflet 지도 라이브러리 문서
- Python 패키지 사용법 조회
- 코드 예제 검색

**출력 형식**:
```
[Context7] 검색: {검색어}
결과:
- {문서 링크}
- {관련 코드 예제}
```

## Sequential Thinking Agent

**역할**: 복잡한 분석 및 설계 전문가

**활성화 조건**:
- "왜", "원인", "분석", "구조", "설계" 키워드
- 인코딩 문제
- 좌표계 변환 로직
- 시스템 아키텍처 검토

**주요 기능**:
- 단계별 문제 분석
- 인코딩 이슈 디버깅
- 좌표계 변환 로직 설계
- 시스템 구조 검토

**출력 형식**:
```
[Sequential] 분석 시작

1단계: {현재 상태 파악}
   └─ {분석 결과}

2단계: {문제 원인 식별}
   └─ {원인 분석}

3단계: {해결책 제안}
   └─ {권장 조치}
```

## OpenAI Codex Agent

**역할**: GPT 기반 코드 검증 전문가

**활성화 조건**:
- "Codex", "GPT", "검증", "리뷰", "대안" 키워드
- 코드 품질 분석 요청
- 버그 탐지 요청
- 성능 최적화 요청

**주요 기능**:
- 코드 품질 분석
- 버그 탐지 및 수정 제안
- 보안 취약점 검사
- 성능 최적화 제안
- 대안 구현 비교

**출력 형식**:
```
[Codex] 코드 검증 결과

품질 점수: {점수}/100

발견된 이슈:
- {이슈 1}: {설명}
- {이슈 2}: {설명}

최적화 제안:
- {제안 1}
- {제안 2}

수정된 코드:
{개선된 코드}
```

## Claude Code Agent

**역할**: 실제 코드 작성 및 실행 전문가

**활성화 조건**:
- "만들어", "수정", "추가", "삭제", "코드" 키워드
- 파일 생성/수정 요청
- 스크립트 실행 요청
- 테스트 실행 요청

**주요 기능**:
- 파일 읽기/쓰기
- 스크립트 실행
- 검증 및 테스트
- Git 작업

**출력 형식**:
```
[Claude Code] 작업 실행

작업: {작업 설명}
파일: {대상 파일}

결과:
{실행 결과}

검증:
✅ {검증 항목 1}
✅ {검증 항목 2}
```

## 에이전트 협업 패턴

### 병렬 실행 (동시에 여러 에이전트)
- 분석 + 문서 검색
- 코드 작성 + 검증

### 순차 실행 (결과를 다음 에이전트에 전달)
- 분석 → 구현 → 검증
- 문서 검색 → 코드 작성 → 테스트

### 하이브리드 (병렬 + 순차 조합)
- (분석 || 검색) → 구현 → (검증 || 테스트)
